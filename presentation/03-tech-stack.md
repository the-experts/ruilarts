# Technologiestack in één oogopslag

- **Backend**: Hono + TypeScript, Neo4j voor matching, PostgreSQL voor opslag, Flask-service voor huisartsdata, Node-geo-service en GraphHopper voor routes.
- **Frontend**: React 19 met TanStack Start, UI via shadcn/Untitled/Radix, formulieren met React Hook Form + Zod, styling in Tailwind.
- **Infrastructuur**: alles draait via Docker Compose, tests met Bruno, code in TypeScript/Node.js 22.

**Waarom dit werkt**
- Snel en type-safe vanaf edge tot database.
- Graphdatabase voor relationele kracht, relationeel voor historie.
- Moderne UI + tooling, klaar voor snelle iteratie en deploy.
